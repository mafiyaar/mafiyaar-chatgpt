# 16 — Open External Actions

External activation requires:

1. Successful `wrangler login`/account authorization.
2. Staging D1 creation and real ID insertion.
3. Real staging workers.dev origin.
4. Staging Worker secrets.
5. Turnstile staging widget/site key/secret.
6. Remote migrations and deployment.
7. Hosted automated tests.
8. Six-person real-room validation.
9. Optional production domain/DNS control.

None of these should be claimed complete without execution evidence.
