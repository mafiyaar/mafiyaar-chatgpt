# MafiYaar Noir-Paper UI System v1

## Direction
A modern social Mafia game with strong noir drama, warm paper surfaces, red danger signals, black authority, and restrained violet utility accents.

## Core visual language
- Warm ivory paper background rather than cute gradients.
- Black, crimson and off-white as the dominant palette.
- Violet only for neutral utilities, spectators and secondary states.
- Large editorial headings and strong hierarchy.
- Rounded interactive cards, but not bubbly or childish.
- Illustrated noir scenes on the key emotional states.
- Public phases feel theatrical; private phases feel deliberate and secret.

## Interaction rules
- Red is reserved for primary progress, elimination, danger and Mafia.
- Purple is reserved for utility, prediction, spectator and neutral secondary actions.
- Every private action keeps the authoritative confirm pattern.
- Server timers and game-state rules remain unchanged.
- The redesign changes presentation only; Cloudflare Worker, D1 and Durable Object logic remain intact.
